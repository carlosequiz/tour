# tour
Tour planner for independent music people.

This project is an Ionic 4 app with Angular 7.

# Getting started
- Install NodeJS, npm
- `npm install`
- Run the dev server with `ionic serve` 

# Used versions

- Node v9.11.1
- npm  5.6.0
- Ionic Framework 4.1.0
- Cordova 9.0.0
- Angular 7.1.4
- AngularFire 5.1.2
- Firebase 5.9.1
